# Word Frequency

Find the word frequency in a text file.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wfq'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wfq

## Usage

From the terminal:

    $ wfq sample.txt
